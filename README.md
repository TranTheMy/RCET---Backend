# RCET-BACKEND

RCET-BACKEND is a Node.js backend service built with Express and MySQL.  
This project provides a clean and scalable backend skeleton to start developing RESTful APIs for the RCET system.

---

## 🚀 Features

- Express.js REST API
- MySQL database integration
- Sequelize ORM
- Clean project structure (controllers, routes, services, models)
- Environment-based configuration
- Ready for authentication, authorization, and module expansion

---

## 🛠️ Tech Stack

- **Node.js**
- **Express.js**
- **MySQL**
- **Sequelize ORM**
- **dotenv**
- **nodemon**
